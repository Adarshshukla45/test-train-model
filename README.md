# 🎯 Student Placement Prediction Model

This project builds a **Machine Learning model** to predict whether a student will get placed or not based on academic and cognitive features like **CGPA, IQ, and performance metrics**.

---

## 🚀 Features

* Predict student placement outcome (Placed / Not Placed)
* Uses Logistic Regression model
* Clean and simple implementation
* Easy to understand for beginners

---

## 🧠 Tech Stack

* Python 🐍
* NumPy
* Pandas
* Scikit-learn
* Matplotlib (for visualization)

---

## 📂 Project Structure

```
placement-project-logistic-regression-main/
│── data.csv
│── model.py
│── train.py
│── README.md
```

---

## ⚙️ How It Works

1. Load dataset
2. Preprocess data
3. Train Logistic Regression model
4. Predict placement outcome

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/test-train-model.git
```

2. Go to project folder:

```
cd test-train-model
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the model:

```
python train.py
```

---

## 📊 Example Output

* Input: CGPA = 8.5, IQ = 120
* Output: ✅ Placed
